添加与删除

unshift：从开头添加
shift：从开头删除
pop：从末尾删除
push：从末尾添加

splice()：删除指定位置指定数量的元素，第三个参数为添加

slice()：删除指定位置指定数量元素

concat：与对象 object.assign 类似，如果是一个数组那么内部所有元素都会被复制，否则将辅助参数本身

遍历

forEach：允许为数组的每个元素都运行一个函数

find：如果返回true则停止搜索并返回该元素，如果没搜索到则返回undefined

findIndex：返回找到的元素的索引，而不是元素本身。如果没找到，则返回 -1。

findLastIndex：方法类似于 findIndex，但从右向左搜索，类似于 lastIndexOf

filter：匹配单个或多个满足条件的元素所组成的数组

map：返回一个全新的满足条件的元素组成的数组

some：检查数组当中是否有满足条件的，有则返回true，否则返回false

排序

sort()：对数组原位排序，更改元素位置也可以指定排序顺序

reverse()：颠倒数组中元素的顺序

split()：分割字符串，返回分割后的数组

join()：将数组粘合成字符串

reduce：将上一次的计算结果作为参数传入下次计算当中，最后汇聚成单个返回值

reduceRight：方法对累加器（accumulator）和数组的每个值（按从右到左的顺序）应用一个函数，并使其成为单个值。

Array.isArray

Array.isArray：判断是否是一个数组，是返回true 否则返回false

Array.from：接受一个可迭代或类数组的值，并从中获取一个“真正的”数组。简单来说就是把伪数组变成可遍历的真数组

特殊性

Map：是一个带键的数据项的集合，就像一个 Object 一样。 但是它们最大的差别是 Map 允许任何类型的键

Set：是一个特殊的类型集合 —— “值的集合”（没有键），它的每一个值只能出现一次。特点是重复使用同一个值调用 set.add(value) 并不会发生改变，这就是 set 里面的每一个值只出现一次的原因

WeakMap & WeakSet

WeakMap：WeakMap 和 Map 的第一个不同点就是，WeakMap 的键必须是对象，不能是原始值。WeakMap 不支持迭代以及 keys()，values() 和 entries() 方法。所以没有办法获取 WeakMap 的所有键或值。

WeakSet：与 Set 类似，但是我们只能向 WeakSet 添加对象（而不能是原始值），跟 Set 一样，WeakSet 支持 add，has 和 delete 方法，但不支持 size 和 keys()，并且不可迭代
